cdelateja Angular

Descripción

Libreria interna de LegoSoft para aplicaciones web echas en Angular.

Formulario básico

El cdelateja cuenta con una clase que es capaz de validar un formulario. Además cuenta con componentes que se integran con la clase para una mejor funcionalidad. Un componente puede tener varias validaciones y por lo tanto puede mostrar todos los errores a la vez.

Login genérico

El cdelateja cuenta con una clase para construir un formulario de autenticación.
